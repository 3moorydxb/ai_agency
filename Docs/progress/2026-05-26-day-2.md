## Website — Full Redesign

### What changed
The first build (Day 1) used Bebas Neue + Montserrat with flat square cards and no animations. Completely replaced the visual layer using the `ui-ux-pro-max` skill (Modern Dark Cinema style).

**Visual system:**
- Fonts: Space Grotesk (headings) + Inter (body)
- Colors: kept existing Nova palette (`#050510`, `#2952D9`, `#6B8FFF`, `#1E3FB8`)
- Cards: glassmorphism — `rgba(255,255,255,0.04)` surface + `rgba(255,255,255,0.08)` border + `backdrop-filter: blur(12px)`
- Buttons: pill-shaped (`border-radius: 999px`) with glow on hover
- Section labels: small ALL-CAPS blue pill badge above every heading

**Hero section:**
- Pill badge with green pulse dot: "Dubai's AI Marketing Agency"
- Gradient text on "AI Marketing" (blue gradient)
- 3 animated ambient orbs in background
- Staggered entrance animations (heroUp keyframe, delays 0.1–0.7s)
- Floating glass stats card: 21 AI Services | 5 Tiers | UAE

**Logo:** Replaced broken circle logo with horizontal logo 2 from `nova_logo.html` — trending-up SVG chart icon + "NOVA." + "Agency · Dubai"

**Layout fixes:**
- Pricing: 5-column grid (all tiers in one row on desktop)
- Builds: flexbox with `justify-content: center` so last row (3 items) centers
- Why Nova: flexbox centered, max-width 380px per card
- Removed USD price conversions (AED only)
- All 5 pricing tiers now use "Get Started" (Scale no longer has two buttons)
