## How to Read This Document

Every service is scored across three factors, each out of 10:

| Factor | What it measures |
|--------|-----------------|
| **Demand** | How much market demand exists in Dubai/UAE. How easy is it to sell? |
| **Ease** | How easy is it to build and run using Claude? 10 = pure AI delivery, 1 = heavy human labor |
| **Payment** | Revenue potential relative to other services (price × recurring value) |
| **Priority** | Combined score — launch order |

High demand + easy to run + decent payment = build first. Services that score 8+ in all three are the ones that keep the business alive while we build out the harder ones.

## Part 3 — Launch Order

### Phase 1 — Launch These First (Priority 8–9)
These are high demand, Claude can run them, and they pay well. Build SOPs for these before anything else.

**Monthly services:**
1. **SEO + GEO + AEO** — Only agency in Dubai doing GEO (ChatGPT) + AEO (Perplexity). Differentiated. AED 4,000/mo.
2. **Social Media Management (2 platforms)** — Highest demand in the market. Pure AI delivery. Easy first sale.
3. **AI Content Engine** — Claude does 100% of this. Blogs, scripts, copy. Easiest to deliver. Entry-level sell.
4. **Paid Ads Management** — Every business with a budget asks for this. Keep in Phase 1 despite needing human oversight — the demand is too high to skip.
5. **Email and SMS Marketing** — Claude writes the sequences. Set up once, runs on autopilot. AED 2,500/mo recurring.

**One-time builds:**
1. **WhatsApp Business Automation Setup** — WhatsApp is how Dubai businesses communicate. Highest demand build.
2. **Chatbot** — Fast to build with Claude Code. Every website needs one.
3. **Website Build** — We already proved this works (novaagency.me). Same workflow for clients.

### Phase 2 — Build Next (Priority 6–7)
These need a bit more process before we can sell them reliably.

**Monthly services:**
- Social media management — 4 platforms
- Personal branding management
- WhatsApp automation management (management after builds are live)
- LinkedIn growth and management
- AI creative production for paid ads

**One-time builds:**
- Workflow automation
- Custom app build
- Shopify Starter Pack + Standard Shopify store
- Booking automation system
- Landing page sprint

### Phase 3 — Build Last (Priority 4–5)
Either low revenue, high labor, or needs the business to be running first.

- Long-form video editing (heavy human labor, skip for now)
- AI image generation (too cheap standalone)
- Voice agents (too complex, too niche for Phase 1)
- AI SDR deployment (needs sales education)
- Influencer sourcing (relationship-dependent)

## Part 4 — How Claude Runs Each Service

The model: **Claude does the work. Worker executes and QCs. Omar teaches the system.**

| Service | Claude's Role | Worker's Role |
|---------|--------------|---------------|
| Social media management | Writes captions, hashtags, calendar, content ideas | Posts on schedule, sends to client for approval |
| AI content engine | Writes blogs, scripts, copy in full | Reviews output, sends to client |
| SEO + GEO + AEO | Writes SEO articles, optimizes for AI engines | Uploads, monitors rankings |
| Email marketing | Writes full sequences, campaign copy | Sets up in Klaviyo/Mailchimp, monitors delivery |
| Paid ads management | Writes ad copy and creatives | Uploads to Meta/TikTok Ads Manager, monitors spend |
| WhatsApp automation | Writes conversation flows and responses | Sets up in WATI/Respond.io, monitors |
| Website build | Claude Code builds the full site | Reviews, QCs, deploys to Netlify |
| Chatbot | Claude Code builds the logic | Tests conversations, deploys |
| Workflow automation | Claude Code + n8n/Make | Tests flows, monitors |
