# Build: Shopify Starter Pack ★

## What the client gets
A bundled package for young ecom clients and first-time store owners: **Shopify store + brand kit + chatbot + 1 month of Launch tier included free**. Fixed price AED 4,500 — no ranges. Maintenance AED 1,500–3,500/mo (Launch tier converts to standard monthly billing in month 2).

## What Worker does — step by step

1. **Onboarding intake — Claude.ai.** Single 30-min call recorded. Drop the transcript (or notes) into Claude.ai with this prompt:
   > "Build a Shopify Starter intake doc for {{CLIENT_NAME}} from this onboarding call transcript. Extract: niche, product list (max 10 SKUs — flag if more, this tier caps), supplier source (AliExpress / Spocket / private supplier / own stock), target market (UAE / GCC / international), brand vibe (3 adjectives + 3 reference brands they love), preferred social platform (IG or TikTok — pick one for the free Launch month), bank/payment provider, domain status (owned / need to buy). Output as `starter-intake-{{CLIENT_SLUG}}.md`. Flag anything missing as `[GAP: ask client before kickoff]`."

2. **Brand kit — speed run.** See `build-brand-identity.md` but compressed (logo, 2 colors, 1 typeface pair, 5-image moodboard). Use Midjourney + Canva.

3. **Shopify store — speed run on Dawn theme.** Cap at 5-10 products. Use Claude to bulk-write product listings (see `build-shopify-store.md` step 4).

4. **Chatbot — basic version.** Shopify Inbox + AI auto-replies OR a basic Tidio install OR Nova's lightweight Claude-API chatbot widget. Pick fastest deploy. Cover: shipping FAQs, return policy, "where's my order".

5. **Launch tier month 1 — see `social-media-management.md` for Launch deliverables.** Set up 1 platform (IG or TikTok per client choice), 10-post calendar, 1 short-form video, 5 product descriptions.

6. **Onboarding handoff — Loom + 1-page doc.** Show client how to add products, edit pages, view orders, read analytics.

## Tools used
